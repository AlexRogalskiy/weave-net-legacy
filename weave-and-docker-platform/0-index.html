Creating distributed applications
with Weave and the Docker platform

This 4-chapter guide is designed to introduce you to how Weave should be used with the Docker platform. As there several complex concepts in such integration, we chose to break it up into small distinct chapters, so a beginner can do it step by step.

An advanced user, however, may wish to jump to either chapter 3 or 4, but we still recommend skimming through the first two chapters.
